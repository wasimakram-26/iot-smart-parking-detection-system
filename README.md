# IoT-Based Smart Parking Detection System

## Overview
This project implements a smart parking detection system using ultrasonic sensors and a microcontroller. The system detects whether a parking slot is occupied or available and indicates the status using LED indicators.

The project demonstrates IoT and embedded system concepts for real-time parking monitoring.

## Features
- Detects vehicle presence using ultrasonic sensor
- Indicates slot availability using LED
- Real-time monitoring system
- Demonstrates sensor interfacing with microcontroller
- Useful for smart parking applications

## Components Used
- Arduino Uno (or ESP32)
- Ultrasonic Sensor (HC-SR04)
- LED indicators
- Resistors
- Breadboard
- Jumper wires

## Working Principle
The ultrasonic sensor measures the distance between the sensor and the object (vehicle). If the distance is below a certain threshold, the system detects that the parking slot is occupied.

The microcontroller processes sensor data and controls LED indicators:
- Green LED → Parking Available
- Red LED → Parking Occupied

## Applications
- Shopping mall parking systems
- Office parking management
- Smart city parking solutions
- Automated parking monitoring
